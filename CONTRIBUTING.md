# Contribuer à Exahia

Merci de votre intérêt pour Exahia ! Voici comment contribuer.

## Comment contribuer

### Signaler un bug
1. Vérifiez que le bug n'a pas déjà été signalé dans les [Issues](https://github.com/Exahia/exahia/issues)
2. Créez une issue avec un titre clair et une description détaillée
3. Incluez les étapes pour reproduire le problème

### Proposer une amélioration
1. Ouvrez une [Discussion](https://github.com/Exahia/exahia/discussions) pour discuter de votre idée
2. Si validée, créez une Pull Request

### Pull Requests
1. Forkez le repo
2. Créez une branche (`git checkout -b feature/ma-feature`)
3. Commitez vos changements (`git commit -m "Ajout de ma feature"`)
4. Poussez (`git push origin feature/ma-feature`)
5. Ouvrez une Pull Request

## Guidelines

- Code en **Python** ou **TypeScript** selon le projet
- Documentation en **français**
- Commits en **français**, concis
- Tests si applicable

## Code de Conduite

Soyez respectueux et constructif. Nous favorisons un environnement inclusif et bienveillant.

## Questions ?

Ouvrez une [Discussion](https://github.com/Exahia/exahia/discussions) ou contactez-nous à admin@exahia.ia.
