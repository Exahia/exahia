# Politique de Sécurité — Exahia

## Signaler une vulnérabilité

Si vous découvrez une vulnérabilité de sécurité dans un projet Exahia, merci de nous contacter directement :

- **Email** : admin@exahia.ia
- **Objet** : [SECURITY] Description courte

Nous nous engageons à :
- Accuser réception sous 48h
- Fournir une évaluation initiale sous 5 jours ouvrés
- Publier un correctif dans les meilleurs délais

## Divulgation Responsable

Nous vous demandons de :
- Ne pas divulguer publiquement la vulnérabilité avant qu'un correctif soit disponible
- Ne pas exploiter la vulnérabilité au-delà de ce qui est nécessaire pour la démontrer
- Ne pas accéder aux données d'autres utilisateurs

## Périmètre

Cette politique couvre tous les repos de l'organisation [Exahia](https://github.com/Exahia).

## Infrastructure de Production

Pour les vulnérabilités liées à l'infrastructure de production (exahia.com), contactez-nous à admin@exahia.ia avec le tag [SECURITY-PROD].
